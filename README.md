# E-commerce Sales Analysis

This project analyzes simulated e-commerce sales data using Python and Pandas.

## Project Structure

analise-vendas-ecommerce/
│
├── dados/
│   └── vendas.csv
│
├── src/
│   └── gerar_dados.py
│
└── README.md

## Technologies

- Python
- Pandas
- Matplotlib

## Dataset

The dataset contains simulated e-commerce sales including:

- sale ID
- date
- product
- customer
- seller
- price
- quantity

## Goals

The objective of this project is to analyze:

- total revenue
- best-selling products
- sales by seller
- sales over time
- top customers
